# Colors

Our color system uses a two-layer architecture: **primitive** raw color scales and **semantic** purpose-based tokens that adapt to light/dark themes.

## Semantic Tokens

These tokens map to primitives and automatically switch between light and dark themes.

### Backgrounds
| Token | Light | Dark | Usage |
|-------|-------|------|-------|
| `--ds-color-bg-primary` | White | #0A0A0A | Main surface |
| `--ds-color-bg-secondary` | #FAFAFA | #141414 | Subtle surface |
| `--ds-color-bg-tertiary` | #F7F7F7 | #1F1F1F | Muted surface |
| `--ds-color-bg-brand` | Coral 50 | Deep coral | Brand tinted |
| `--ds-color-bg-glass` | rgba white 72% | rgba black 72% | Glassmorphism |

### Foregrounds
| Token | Light | Dark | Usage |
|-------|-------|------|-------|
| `--ds-color-fg-primary` | #141414 | #F7F7F7 | Primary text |
| `--ds-color-fg-secondary` | #545454 | #AFAFAF | Secondary text |
| `--ds-color-fg-brand` | Coral 500 | Coral 400 | Brand text |
| `--ds-color-fg-link` | Coral 500 | Coral 400 | Interactive links |

### Actions
| Token | Light | Dark | Usage |
|-------|-------|------|-------|
| `--ds-color-action-primary` | Coral 500 | Coral 500 | Primary CTA |
| `--ds-color-action-secondary` | #141414 | White | Secondary CTA |
| `--ds-color-action-danger` | Rose 500 | Rose 500 | Destructive |
| `--ds-color-action-success` | Emerald 500 | Emerald 500 | Positive |

### Gradients
| Token | Usage |
|-------|-------|
| `--ds-color-gradient-brand` | Primary brand gradient (buttons, progress) |
| `--ds-color-gradient-sunset` | Warm decorative gradient |
| `--ds-color-gradient-ocean` | Cool decorative gradient |
| `--ds-color-gradient-glass` | Glassmorphism overlay |
